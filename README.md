# 1: Advanced Image Classification using CIFAR-10

## 🚀 Project Overview
This project focuses on building an advanced image classification model using the CIFAR-10 dataset. The model is trained to classify images into 10 different categories using deep learning techniques, specifically Convolutional Neural Networks (CNN).

---

## 📂 Dataset
The CIFAR-10 dataset consists of:
- 60,000 color images (32x32 pixels)
- 50,000 training images
- 10,000 testing images

### Classes:
- Airplane  
- Automobile  
- Bird  
- Cat  
- Deer  
- Dog  
- Frog  
- Horse  
- Ship  
- Truck  

---

## 🧠 Model & Approach
- Implemented Convolutional Neural Network (CNN)  
- Data preprocessing and normalization  
- Model training with optimized parameters  
- Used dropout to reduce overfitting  

---

## ⚙️ Technologies Used
- Python  
- TensorFlow / Keras  
- NumPy  
- Matplotlib  

---

## 📊 Results
- Achieved good classification accuracy on test data  
- Visualized training and validation performance using graphs
